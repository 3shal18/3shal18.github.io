Java Shopping Cart Simulation
Java GUI app that simulates an online store where users can search for items, add them to a cart, check out, and get an invoice. The program reads inventory from a CSV file, updates totals with tax/discounts, and logs each transaction in a separate file. The program manages inventory by reading item data (IDs, descriptions, prices, quantities) from a CSV file.

Skills learned:

  • Problem-solving through java programming
  • How to design user-friendly applications
  • How to work with structured data (CSV files)
  • Simulating real-world business logic in code
