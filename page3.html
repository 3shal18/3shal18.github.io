Java Shopping Cart Simulation

- Built a Java GUI app that simulates an online store where users can search for items, add them to a cart, check out, and get an invoice.
- Program reads inventory from a CSV file, updates totals with tax/discounts, and logs each transaction in a separate file.
- Integrated inventory management by reading item data (IDs, descriptions, prices, quantities) from a CSV file.
- Implemented discounts, tax calculations, and dynamic invoice generation at checkout.
